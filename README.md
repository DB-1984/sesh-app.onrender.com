# SESH | Fitness & Workout Diary

## Shadcn, Express, React/RTK Query, Mongo DB

SESH is a workout diary built using a React frontend in combination with RTK querying, an Express backend, and a MongoDB database layer. The interface is minimalist, featuring two 'dual-mode' layouts alongside a user profile section and date filtering. The UI is engineered using Tailwind CSS and shadcn/ui components.
